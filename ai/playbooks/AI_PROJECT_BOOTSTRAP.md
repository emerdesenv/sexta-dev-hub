# AI Project Bootstrap

Este playbook orienta a criação de novos projetos utilizando o AI Engineering Framework.

---

# 1. Definir objetivo do projeto

Identificar:

- problema a ser resolvido
- público alvo
- principais funcionalidades

---

# 2. Definir stack tecnológica

Selecionar:

- frontend
- backend
- banco de dados
- infraestrutura

---

# 3. Criar estrutura inicial

Definir estrutura básica do projeto.

Consultar:

/ai/CONTEXT/PROJECT_MAP.md  

---

# 4. Definir arquitetura

Consultar:

/ai/CONTEXT/ARCHITECTURE.md  

Definir:

- organização de módulos
- comunicação entre serviços
- responsabilidades das camadas

---

# 5. Definir padrões de código

Consultar:

/ai/GOVERNANCE/CODE_STYLE.md  

Definir convenções para:

- nomenclatura
- organização de arquivos
- padrões de implementação

---

# 6. Configurar governança

Consultar:

/ai/GOVERNANCE/AI_RULES.md  
/ai/GOVERNANCE/CONSTRAINTS.md  

Definir regras do projeto.

---

# 7. Configurar testes

Consultar:

/ai/ENGINEERING/TESTING_GUIDELINES.md  

Definir estratégia de testes desde o início.

---

# 8. Iniciar desenvolvimento

Utilizar:

- agentes especializados
- workflows do framework
- playbooks disponíveis
