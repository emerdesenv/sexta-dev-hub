# AI Setup Guide

Guia de onboarding humano para usar o framework documental de IA deste projeto.

---

# 1. Comece por aqui

Leia nesta ordem:

1. `ai/README_AI.md`
2. `ai/ACTIVATE_PROJECT_CONTEXT.md`
3. `ai/governance/DOCUMENT_PRECEDENCE.md`
4. `ai/AI_SYSTEM_MAP.md`

---

# 2. Como usar no dia a dia

- Antes da tarefa: aplique `ai/ACTIVATE_PROJECT_CONTEXT.md`.
- Para decidir fluxo: use `ai/workflows/AI_TASK_ROUTER.md`.
- Para tarefas complexas: use `ai/workflows/AI_AGENT_ORCHESTRATOR.md`.
- Para execução guiada: use playbooks em `ai/playbooks/`.
- Para validação final: use `ai/governance/DELIVERY_CHECKLIST.md`.

---

# 3. Ferramentas

Compatível com ferramentas que conseguem ler arquivos do repositório (Cursor, ChatGPT, Copilot, Claude e similares).

---

# 4. Observação importante

Em caso de conflito entre documentos, siga `ai/governance/DOCUMENT_PRECEDENCE.md`.
