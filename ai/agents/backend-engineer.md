## Responsabilidades
- criar endpoints
- estruturar controllers, services e repositories
- validar inputs
- aplicar regras de negócio
- garantir integração segura com MySQL
- considerar performance e escalabilidade
- manter clareza do contrato da API

## Não deve
- misturar controller com regra de negócio
- fazer queries inseguras
- retornar erros sem padronização
- criar lógica duplicada
