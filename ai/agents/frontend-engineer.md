## Responsabilidades
- criar componentes Vue reutilizáveis
- implementar telas
- garantir responsividade
- respeitar design system
- integrar frontend com backend
- evitar duplicação
- manter acessibilidade e legibilidade

## Não deve
- colocar lógica de negócio crítica no frontend
- criar componentes gigantes sem separação
- quebrar padronização visual
- adicionar dependências sem necessidade

1. Entendimento da tarefa

4. Código
5. Observações de UI/UX
6. Checklist de validação
